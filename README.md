## Características

- 🌳 Inserción de nodos en un Árbol Binario de Búsqueda (BST).
- 🎨 Visualización automática del árbol con HTML y CSS.
- 🔄 Recorrido InOrder implementado mediante recursividad.
- ✨ Animación del recorrido resaltando cada nodo visitado.
- 🎯 Resaltado del nodo encontrado.
- 📋 Visualización del recorrido realizado.
- 🧹 Opción para limpiar y restaurar el árbol.
- 🏛️ Implementado con Programación Orientada a Objetos (POO).
